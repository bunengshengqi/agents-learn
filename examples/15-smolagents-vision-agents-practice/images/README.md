# Day15 Images

Put real images here when you want the Day15 tools to call a vision model.

Suggested paths:

```text
images/guests/guest_claim_wonder_woman.png
images/screenshots/party_signup_error.png
```

The file stem is the name used in `first_agent.py`. For example,
`guest_claim_wonder_woman.png` can be referenced as `guest_claim_wonder_woman`.
