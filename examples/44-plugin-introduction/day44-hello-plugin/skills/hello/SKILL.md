# Hello from Day 44

When the user asks for a greeting or invokes this skill:

1. Greet the learner warmly in the language they used.
2. Explain in one sentence that this skill is one component bundled inside `day44-hello-plugin`.
3. Give the analogy: the plugin is a toolbox, while this skill is one instruction card inside it.
4. Keep the response under 120 words unless the user asks for more detail.

Do not claim that this teaching plugin includes an MCP server, connector, or hook.
